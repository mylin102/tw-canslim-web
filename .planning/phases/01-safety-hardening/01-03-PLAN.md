Supported operational publishes must go through one bundle call:

```python
publish_artifact_bundle(
    {
        "docs/data.json": data_payload,
        "docs/data_light.json": light_payload,
        "docs/update_summary.json": summary_payload,
    },
    lock_path="docs/.publish.lock",
)
```

Rollback must restore the publish bundle, not a single file:

```python
restore_latest_bundle(targets=(
    "docs/data_base.json",
    "docs/data.json",
    "docs/data_light.json",
    "docs/data.json.gz",
    "docs/update_summary.json",
))
```

Unsupported legacy direct writers must exit before touching live `docs/` artifacts and point maintainers to the supported scripts/workflows.

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| supported maintenance scripts→publish bundle | Operational scripts can overwrite the live GitHub Pages artifact set |
| workflow trigger→repo publish surface | Scheduled and issue-triggered Actions runs can overlap unless serialized |
| maintainer→rollback CLI | Recovery commands can restore or mis-restore production artifacts |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-09 | T | supported quick/batch/on-demand writers | mitigate | Force supported operational writers through `publish_artifact_bundle(...)` with one repo-level lock. |
| T-01-10 | R | operational failure reporting | mitigate | Publish explicit retry/failure summaries and replace silent suppression with logged exceptions. |
| T-01-11 | D | GitHub Actions overlap | mitigate | Add workflow concurrency groups to both scheduled and on-demand workflows for the shared publish surface. |
| T-01-12 | E | legacy direct writers | mitigate | Deprecate unsupported scripts so default execution cannot write live `docs/` artifacts. |
| T-01-13 | T | rollback CLI | mitigate | Restore only manifest-backed validated bundles and exit non-zero on restore failure. |
</threat_model>
