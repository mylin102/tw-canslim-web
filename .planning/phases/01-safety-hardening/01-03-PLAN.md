| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-01-10 | R | `verify_local.py` | mitigate | Replace bare exception with explicit logged failure so fallback behavior is traceable. |
| T-01-11 | D | rollback CLI | mitigate | Restrict CLI to known target files and require validated backup selection before restore. |
| T-01-12 | E | restore path | mitigate | Exit non-zero on restore failure so broken recovery cannot masquerade as success. |
