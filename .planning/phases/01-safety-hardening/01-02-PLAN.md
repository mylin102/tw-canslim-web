<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| external API/parquet input→exporter | Remote or cached market data becomes published static JSON |
| exporter→`docs/data.json` | Main publish path overwrites the GitHub Pages artifact read by users |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-05 | T | `export_canslim.py` resume/load path | mitigate | Validate existing JSON before resume and discard corrupt payloads with explicit warning. |
| T-01-06 | D | `export_canslim.py` concurrent publish | mitigate | Route both incremental and final saves through locked helper writes. |
| T-01-07 | R | `export_dashboard_data.py` error handling | mitigate | Replace return-only failure on missing input with raised exception plus log entry. |
| T-01-08 | E | published JSON contract | mitigate | Force `schema_version=1` and helper validation so later readers cannot accidentally consume malformed structures. |
</threat_model>
