## 🎯 實作計畫

### Phase 1: 整合 TEJ 到 CANSLIM Engine (20 行)
- 在 `export_canslim.py` 中引入 `tej_processor.py`
- 用 TEJ 月營收資料計算 **C 指標** (季營收成長率)
- 用 TEJ 損益表季資料計算 **A 指標** (EPS 年成長率)
- 用 TEJ 基本資料取代原本簡易的產業分類

### Phase 2: 前端顯示 TEJ 財務數據 (30 行)
- 個股詳情頁新增「EPS 走勢圖」
- 顯示近 4 季 EPS 和營收
- 標註 C、A 指標的計算結果

### Phase 3: 自動排程 (15 行)
- GitHub Actions 每日透過 TEJ 更新資料
- 取代 yfinance 和 FinMind（覆蓋率更好）
